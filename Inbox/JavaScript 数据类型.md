# JS 数据类型

[[JavaScript 数据类型|数据类型]]指的就是[[JavaScript 字面量|字面量]]的类型, 在 js 中一共有六种数据类型  
  
#### 前五种是基本数据类型:  
- [[String 字符串]] 
- [[Number 数字 ]] 
- [[Boolean 布尔值]]  
- [[Null 空值]]  
- [[Undefined 未定义]]  
  
#### 第六种是引用数据类型  
- [[Object]] 对象
