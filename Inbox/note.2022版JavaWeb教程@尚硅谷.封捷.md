## 03. HTML 中的基础标签

- html 页面由一对标签组成, `<html></html>`
	- <html> 称之为开始标签
		- </html> 称之为结束标签

```html
<html>
	<head>
		<title>网页的标题</title>
	</head>
	<body>
		Hello World !
	</body>
</html>
```
