# Boolean 布尔值

- Boolen 布尔值  
- 布尔值只有两个: 主要做逻辑判断  
	- true  
		- 表示真  
	- false  
		- 表示假  
- 使用 typeof 检查一个布尔值时, 会返回 boolean

```js
var bool = true;  
console.log(bool);  // true
console.log(typeof bool); //boolean
```
