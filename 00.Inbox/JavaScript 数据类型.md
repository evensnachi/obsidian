# JS 数据类型

[[JavaScript 数据类型|数据类型]]指的就是[[JavaScript 字面量|字面量]]的类型, 在 js 中一共有六种数据类型  
  
### 前五种是基本数据类型:  
- [[String 字符串]] 
- [[Number 数字 ]] 
- [[Boolean 布尔值]]  
- [[Null 空值]]  
- [[JavaScript Undefined 未定义]]  
  
### 第六种是引用数据类型  
- [[Object]] 对象

## 强制类型转换

- 强制类型转换  
	- 指将一个数据类型前置转换为其他的数据类型.  
	- 类型转换主要指, 将其他数据类型, 转换为:  
		- String 
		- Number 
		- Boolean
